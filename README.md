# todoapp

##Простое приложение для ежедневных целей, сделанное на Vue / Vuex
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
